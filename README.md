# DSL -   a simple scala DSL code to calculate  different current expression
    
 ```scala   
     5(US)*5 + 3 (POU)/2 - 2 (RMB) > RMB

     (5 | US)*5 + (3 | EUR)/2 - 5 (RMB) > RMB

     (5 | US)*10 + 3 (EUR) > RMB 
 ```
